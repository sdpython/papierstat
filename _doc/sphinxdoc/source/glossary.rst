
.. index:: glossary

Glossary
========

.. glossary::

    cheatsheets

        Mot-clé très populaire sur internet pour désigner un pense-bête
        avec les bouts de code ou formules les plus utilisés.
        Exemple : `cheat sheets <http://www.xavierdupre.fr/app/ensae_projects/helpsphinx/cheat_sheets.html>`_,
        `Anti-sèches <http://www.xavierdupre.fr/app/ensae_teaching_cs/helpsphinx/antiseches/index.html>`_
        (machine learning).

    cible

        La *cible* est ce qu'il faut prédire, ce vers quoi un modèle
        de machine learning supervisé doit tendre.

    features

        Traduction anglaises de *variables*.

    Jupyter
        See :epkg:`Jupyter`

    pandas
        See :epkg:`pandas`.

    scikit-learn
        See :epkg:`scikit-learn`.

    target

        Traduction anglaise de *cible*.

    variable

        Les modèles de machine learning sont des fonctions qui prennent le plus
        souvent comme entrées un vecteur numérique souvent nommé :math:`X`.
        Chaque dimension du vecteur
        vient d'une base de données ou est calculée à partir de cette base.
        On appelle chacune d'elles une *variable*. Les variables
        sont les entrées du modèle.
