    *Séries temporelles*

    Le module standard est :epkg:`statsmodels` qui implémente
    tous les modèles linéaires associés aux séries temporelles.
    Récemment, le module :epkg:`prophet` l'a supplanté avec plus
    d'options pour prédire, extraire la tendance, lissé,
    extraire la saisonnalité. Une approche assez nouvelle consiste
    à décomposer la série pour extraire les changements de tendance :
    changement de tendances, le module peut tenir compte
    des jours fériés.
